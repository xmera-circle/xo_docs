Fehler in der Dokumentation melden

Wenn Du in einem der Handbücher einen Fehler findest oder ein Thema fehlt, freuen wir uns über eine Fehlermeldung von Dir.

Sich an der Dokumentation zu beteiligen ist sehr einfach, da der gesamte Code auf GitHub verfügbar ist und dort bearbeitet werden kann.

Fehlermeldungen kannst Du direkt auf GitHub oder auf unserer Entwicklungsplattform xmera Circle einstellen. xmera Omnia Nutzer mit Subskription können eine Fehlermeldung über das Serviceportal an uns richten.
