Fehler in der Dokumentation melden

Wenn Sie in einem der Handbücher einen Fehler finden oder ein Thema fehlt, freuen wir uns über eine Fehlermeldung von Ihnen.

Sich an der Dokumentation zu beteiligen ist sehr einfach, da der gesamte Code auf GitHub verfügbar ist und dort bearbeitet werden kann.

Fehlermeldungen können Sie direkt auf GitHub oder auf unserer Entwicklungsplattform xmera Circle einstellen. xmera Omnia Nutzer mit Subskription können eine Fehlermeldung über das Serviceportal an uns richten.
